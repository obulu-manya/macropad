# Macropad

A compact PCB design created using **KiCad** as part of my PCB design learning journey. This project covers the complete process from schematic design to PCB layout and preparing files for PCB manufacturing.

---

## About

The goal of this project was to understand the complete PCB design workflow using KiCad. It includes creating the schematic, designing the PCB layout, routing the board, checking for design errors, and generating manufacturing files.

This project helped me gain practical experience in designing a PCB from start to finish.

---

## Travel Purpose

A compact macropad can be a useful travel companion for people who work on the go. It can be used to create custom shortcuts for frequently used applications, making everyday tasks faster while travelling with a laptop or tablet.

Its portable size also makes it suitable for remote work, study, and travel setups where desk space is limited.

## Project Workflow

- Created the schematic
- Designed the PCB layout
- Routed the PCB traces
- Performed Design Rule Check (DRC)
- Generated Gerber files for manufacturing

## Project Files

### Design Files


### Gerber Files

```text

macropad obulu-B_Cu.gbr

macropad obulu-B_Paste.gbr
macropad obulu-F_Silkscreen.gbr
macropad obulu-B_Silkscreen.gbr

macropad obulu-job.gbrjob
```
